# Data Description

## Overview

This folder contains the metadata file describing all 123 indicators used in the nowcasting study. The raw macroeconomic panel is **not included** due to institutional data agreements.

---

## Files in this folder

### `meta_data_v2.csv` — Indicator metadata (included)

| Column | Description |
|---|---|
| `series` | Variable code (matches column names in `data_tf.csv`) |
| `name` | Variable description in Spanish |
| `freq` | Frequency: `M` = monthly, `Q` = quarterly |
| `block_g` | DFM block: Global/Real activity (1 = yes, 0 = no) |
| `block_s` | DFM block: External sector |
| `block_r` | DFM block: Financial/monetary conditions |
| `block_l` | DFM block: Labour/fiscal |
| `months_lag` | Publication lag in months (0 = available same month, 1 = 1 month delay, etc.) |

### `data_tf.csv` — Transformed panel (NOT included)

Generated by running `Models/prepare_data_v2.R` on the raw data.
Contains month-on-month growth rates for all 123 indicators, 2005M1–2024M6.

### `SLV_nowcasting_data_mq.csv` — Raw data (NOT included)

Original mixed-frequency panel before transformation.

## How to Obtain the Raw Data

### El Salvador official sources

| Institution | Variables | URL |
|---|---|---|
| BCR (Banco Central de Reserva) | IVAE, monetary aggregates, credit, interest rates, fiscal | www.bcr.gob.sv |
| DIGESTYC | National accounts GDP (quarterly) | www.digestyc.gob.sv |
| ISSS (Instituto Salvadoreño del Seguro Social) | Employment by sector | www.isss.gob.sv |
| Ministerio de Hacienda | Tax revenues, government expenditure | www.transparenciafiscal.gob.sv |

## Variable Naming Convention

Variables follow the naming convention used by the BCR:

- `IVAE_TOT` — IVAE total (economic activity index)
- `IVAE_AG` — IVAE agriculture sector
- `GDP` — Quarterly GDP growth rate (QoQ)
- `GDP_AG` — GDP agriculture sub-sector
- `CE_TOT_VV` — Total commerce, nominal
- `EXP_CO` — Exports, coffee
- `REM` — Remittances
- `TIP_30` — 30-day lending rate
- `INGT_IVA` — VAT tax revenues
- `UNEM_US` — US unemployment rate
- `EFFR_US` — US effective federal funds rate

## Transformation Details

All series are transformed to month-on-month growth rates using:

```
growth_rate(t) = (x(t) / x(t-1)) - 1
```

Exceptions:
- Interest rates: first differences (percentage point changes)
- Survey indices: levels
- Series already in rate/index form: levels or first differences as appropriate

See `Models/prepare_data_v2.R` for the complete transformation pipeline.
